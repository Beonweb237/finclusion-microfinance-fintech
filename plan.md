# Plan — Microfinance & Fintech Premium Website Template (Sector 10)

## Overview
Build a premium, market-ready React website template for Microfinance & Fintech organizations. The template targets underbanked individuals, TPE/SMEs, diaspora, and impact investors. Key features include a credit simulator, impact dashboard, 3-step process flow, and dual CTA paths (borrower + investor).

## Stage 1: Design & Setup
- **Skill**: vibecoding-webapp-swarm (webapp-building-swarm)
- **Input**: Sector analysis from `sector-10-fintech.md` + detailed guide from `10_microfinance_fintech.md`
- **Output**: Design system, component architecture, project scaffold
- **Tasks**:
  1. Set up Next.js + Tailwind + shadcn/ui project scaffold
  2. Define design system (colors, typography, spacing)
  3. Create reusable component library (nav, cards, forms, footer)
  4. Generate visual assets (hero images, team portraits, testimonial photos)

## Stage 2: Public Pages Development
- **Skill**: vibecoding-webapp-swarm
- **Output**: All 8 public-facing pages
- **Pages**:
  1. **Homepage** — Hero + products + credit simulator + 3-step process + impact numbers + testimonials + CTA
  2. **Products** — Credit, savings, transfers, payments, micro-insurance
  3. **How It Works** — 3-step registration & first use process
  4. **Impact** — Impact numbers, client stories, institutional partners
  5. **About** — Mission, team, partners, governance, pricing transparency
  6. **Security** — Compliance, data protection, certifications
  7. **Blog** — Financial advice, inclusion news
  8. **Contact** — Support, partnerships, press

## Stage 3: User Space & Admin Dashboard
- **Skill**: vibecoding-webapp-swarm
- **Output**: User portal + Admin dashboard
- **User Space**: Simplified client dashboard (accounts, transactions, loan requests, documents)
- **Admin Space**: Loan requests, client management, content management

## Stage 4: Polish & Deploy
- **Output**: Deployed website
- **Tasks**:
  1. SEO metadata and schema markup
  2. Accessibility checks (WCAG AA)
  3. Responsive testing
  4. Animation refinements
  5. Deploy to production

## Design Direction
- **Palette**: Green inclusion (#10B981) + White (#FFFFFF) + Warm accents (#F59E0B amber)
- **Typography**: Geist Sans (headings) + Geist Mono (labels/data)
- **Mood**: Inclusive, simple, fast, secure, warm
- **Style**: Light template with generous whitespace, rounded cards (14px radius)
